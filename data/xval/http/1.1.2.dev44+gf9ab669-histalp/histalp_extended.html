Extended mass balance calibration for the Alpine reference glaciers

Here you can find results for the cross-validation when the following model parameters are varied:

  • Precipitation Factor
  • Melt Temperature
  • Liquid Precipitation Temperature
  • Temperature Lapse Rate
Each possible parameter combination (1215) is cross-validated and plotted with respect to one of the parameters. The box-whisker plots show the full range of results and the red dots indicates the results when using the proposed standard values.

All results for the Alpine reference glaciers are calculated with the ZAMG Histalp data set.

It is sensible to use different model parameters for regional model runs with different climate data. You can find a extended discussion about the proposed model parameters when using the ZAMG HISTALP data set on the OGGM blog.
