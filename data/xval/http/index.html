Mass Balance calibration continuous integration

This website tracks the performance of the OGGM mass balance calibration across multiple model versions. This allows us to easily see the effects of certain model changes on the mass balance parametrisation of OGGM.


How to use:
Currently there are four different types of cross-validations to look at:

  • Cross-validations for all global reference glaciers (baseline climate data: CRU)
  • Cross-validations for the Alpine reference glaciers (baseline climate data: HISTALP)
  • A short cross-validation with the standard model parameters
  • A extended cross-validation with varying model parameters
For each setting (which you can reach via the navigation bar) you will find several OGGM model versions to switch between. In this way it is possible to immediately see if a particular model update changed the mass balance calibration.
